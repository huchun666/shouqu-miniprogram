# 如何获取图标资源

## 🚀 最快解决方案（推荐）

### 方法1：使用在线占位符生成器

1. 打开浏览器，访问：https://via.placeholder.com/
2. 使用以下格式生成图标：
   - TabBar图标：`https://via.placeholder.com/81x81/1a1a1a/ffffff.png`
   - 功能图标：`https://via.placeholder.com/48x48/1a1a1a/ffffff.png`
3. 右键保存图片到 `assets/icons/` 目录
4. 重命名为对应的文件名

### 方法2：使用图标库（推荐用于生产环境）

#### iconfont（阿里巴巴图标库）
1. 访问：https://www.iconfont.cn/
2. 注册/登录账号
3. 搜索需要的图标（如：首页、设置、电池等）
4. 添加到项目
5. 下载PNG格式
6. 统一尺寸后保存到 `assets/icons/` 目录

#### Icons8
1. 访问：https://icons8.com/
2. 搜索图标
3. 选择PNG格式下载
4. 调整尺寸后保存

### 方法3：手动创建简单图标

使用任何图片编辑软件（Photoshop、GIMP、甚至系统自带的画图工具）：

1. 创建新图片，尺寸如下：
   - TabBar图标：81px × 81px
   - 功能图标：48px × 48px
   - 小图标：32px × 40px
   - 头像：120px × 120px

2. 填充纯色背景（如 #1a1a1a）
3. 添加简单图形或文字
4. 导出为PNG格式
5. 保存到 `assets/icons/` 目录

## ✅ 验证步骤

下载/创建图标后，请验证：

1. **检查文件是否存在**
   ```bash
   ls -la assets/icons/
   ```

2. **检查文件名是否正确**
   - 文件名必须与代码中引用的完全一致
   - 区分大小写
   - 扩展名必须是 `.png`

3. **检查文件路径**
   - 代码中使用：`/assets/icons/xxx.png`
   - 实际文件路径：`assets/icons/xxx.png`

## 🔧 临时解决方案

如果急需运行项目，可以：

1. **使用纯色方块**
   - 创建纯色PNG图片
   - 保存为对应文件名

2. **使用emoji图标**
   - 修改代码使用emoji代替图片
   - 例如：`<text>🔓</text>` 代替 `<image src="...">`

3. **注释掉图标**
   - 临时注释掉图标显示
   - 仅显示文字

## 📝 注意事项

1. **图标尺寸**
   - TabBar图标建议提供@2x和@3x版本
   - 功能图标使用单倍图即可

2. **图标风格**
   - 建议使用统一的图标风格
   - 颜色建议使用主题色 #1a1a1a

3. **文件格式**
   - 必须使用PNG格式
   - 支持透明背景

4. **文件命名**
   - 必须与代码中引用的名称完全一致
   - 区分大小写

## 🆘 遇到问题？

1. 检查文件路径是否正确
2. 检查文件名是否完全匹配（包括大小写）
3. 检查图片格式是否为PNG
4. 查看微信开发者工具的控制台错误信息

如果问题仍未解决，请检查：
- `app.json` 中的 tabBar 配置
- 页面中的图片路径引用
